# LiteGUI Guides

LiteGUI allows you to create powerful and customizable interfaces using Javascript (or TypeScript) without checking HTML manuals. It mimics a desktop application look and feel.

It is designed for web-apps where the interface changes dynamically and needs to handle many widgets efficiently.

## Getting Started

1. Check the [Installation and Basics](litegui.md).
2. Learn how to build a [Full Application Layout](building_layout.md).

## Core Components

- **Inspector**: The most important class, used to create property editors (sliders, buttons, text inputs). [Read Guide](inspector.md).
- **Area**: To handle split views and flexible layouts. [Read Guide](area.md).
- **Dialog**: Floating windows for alerts, settings, or tools. [Read Guide](dialog.md).
- **Menubars**: Top level menu for your application.
- **Tabs**: To organize content in tabs. [Read Guide](tabs.md).
- **ContextMenu**: Right-click menus. [Read Guide](contextmenu.md).
- **Table**: To display data in rows and columns. [Read Guide](table.md).

## Helper Methods

The `LiteGUI` global object contains methods for:
- Dialogs (Alert, Confirm, Prompt)
- HTTP Requests
- DOM manipulation
- Clipboard handling

See the [LiteGUI Guide](litegui.md) for more details.
